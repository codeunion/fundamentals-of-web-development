## Introduction to JavaScript

Goals:

1. Learn syntax in relation to contextualized Ruby (using select exercises)
  - Data types
  - Functions
  - Data structures
1. Explore the primary execution context of JavaScript (i.e. a web browser)
  - Loading JS files
  - The console
  - Traversing HTML
  - Debugging
