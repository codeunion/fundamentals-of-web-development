Modal Dialog Demo

This is a demo of a 100% home-grown modal dialog system. Click the buttons below to see the modal dialogs. View the source code for this page to see the nitty gritty.

Relevant Files

Please, please, please look at the contents of these files.

modal.css

modal.css contains all the styling for the modal dialogs. The key idea is to use "position: fixed" and "z-index" to position the modal dialogs above every other element on the page, preventing the user from interacting with any element "below" the modal dialog.

modal.js

modal.js contains all the behavior logic for the modal dialogs. The key idea is to assume as little as possible about the underlying DOM structure, i.e., what elements we need, what classes they should have, and how they're arranged in the document relative to each other.

Email Modal

To close this dialog, you can do one of three things:

  1. Click the "X" button

  2. Click the semi-transparent background surrounding the modal dialog

  3. Press the ESC key.

See modal.js for the JavaScript code that drives these behaviors.
